# YogaGuruji.github.io
About
This project is a modern, responsive landing page for Yoga Guruji – Rahul, Yoga & Fitness Trainer from Delhi. It is built with pure HTML, Tailwind CSS (via CDN), and a bit of JavaScript for theme toggling, animations, and contact form handling.

The website is designed to help Rahul:

Showcase his online morning yoga batches and offline classes in Delhi.

Highlight pain‑relief focused programs (cervical, neck, back, posture, eyesight) and general fitness.

Convert visitors from Instagram (@yogagurujii) and YouTube (Yoga Guruji – @yogaguruji4404) into leads via WhatsApp, call, and contact form.

Key features:

Hero section focused on transformation: body, mind, and lifestyle.

“Programs” section for online batches, offline yoga & fitness, pain‑relief programs, and corporate/group sessions.

Dedicated YouTube section linking to the Yoga Guruji channel with embedded video.

Student testimonials to build trust and social proof.

Instagram follow section pointing to @yogagurujii.

Contact area with Call/WhatsApp (+91 95404 05580) and an EmailJS‑ready form (you just plug in your own keys).

Dark/light mode toggle and smooth fade‑up animations for a clean UI/UX.

You can fork this repo and customize:

Text content (batch timings, pricing, location details).

YouTube video ID in the embed.

EmailJS public key, service ID, and template ID for production contact form.

Colors, logo image, and fonts to match your own yoga/fitness brand.
